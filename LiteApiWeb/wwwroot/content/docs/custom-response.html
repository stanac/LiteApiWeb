Custom status, headers and response

Since release 0.8 it's possible to return any type of response from action with any custom content, response code, and header. This is done by using following methods available in any controller:

  • ILiteActionResult Content(string data, string contentType)
  • ILiteActionResult Content(byte[] data, string contentType)
  • ILiteActionResult Json(string jsonContent)
  • void SetResponseStatusCode(int? responseCode)
  • void AddResponseHeader(string key, StringValues values)
  • void AddResponseHeaders(IDictionary<string, StringValues> keyValuesPairs)

See following sample action:

public class SomeController: LiteController
{
    public ILiteActionResult SomeResponse()
    {
        AddResponseHeader("key", "val");
        SetResponseStatusCode(241);
        return Json("{ \"a\": \"b\" }");
        // or
        // return Content("response body", "text/plain");
    }
}
