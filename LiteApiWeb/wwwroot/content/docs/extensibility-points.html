Extensibility points

Following extensibility points are available:

  • Custom query model binder
  • Custom policy authorization
  • Custom authorization filter
  • Custom response (low level) (ILiteActionResult)
  • Custom response, content-type, header, status code
  • Replacing any internal service with custom implementation
  • Global filters
  • For virtual methods BeforeActionExecution and AfterActionExecuted please check LiteController page
